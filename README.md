# Welcome to the Agentic App in Fabric Workshop!
![Watch the Demo: SQL Agentic App on Fabric](./assets/header2.png)
## Agenda

In this hands-on workshop, you will learn about:

- Integrating Agentic Application to Analytics (E2E)
- Learning about Fabric workloads in the context of AI generated data
  - Operational Database
  - Data science workloads
  - BI and Analytics
  - Streaming and Real-time alerting
  - Data Agent
- Building a common data model for AI
- Creating a Fabric data agent
  - Preparing semantic model for data agent consumption
  - Conguring & evaluating data agent
